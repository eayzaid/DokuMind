# DokuMindEntreprise

[cite_start]**DokuMindEntreprise** is a SaaS Platform for Intelligent Knowledge Management[cite: 3]. [cite_start]It allows organizations to centralize their internal information and interact with it through a secure, document-based assistant[cite: 7, 12].

> [!IMPORTANT]
> [cite_start]**Status:** This project is currently **in development**[cite: 1, 6]. [cite_start]Final features and acceptance criteria may evolve based on ongoing research[cite: 6].

---

## 🌟 Core Functionalities

* [cite_start]**Isolated Workspaces**: Each organization registers for a dedicated environment to ensure its data remains private and secure[cite: 9, 12].
* [cite_start]**Knowledge Base Ingestion**: Authorized users can upload PDF documents to build a custom library of company information[cite: 12].
* [cite_start]**Smart Assistant**: Employees can ask questions in natural language and receive answers based specifically on their company's documents[cite: 12].
* [cite_start]**Reliable Responses**: The system is designed to manage hallucinations by only answering from provided context and stating when information cannot be found[cite: 12].
* [cite_start]**Role-Based Access**: Multi-level permissions (Admin, Manager, Curator, and Employee) control who can manage users or upload documents[cite: 9, 12].
* [cite_start]**Strict Data Isolation**: An infrastructure designed to ensure that one organization can never access the data of another[cite: 12].
