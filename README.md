# DokuMindEntreprise

**DokuMindEntreprise** is a SaaS Platform for Intelligent Knowledge Management.It allows organizations to centralize their internal information and interact with it through a secure, document-based assistant.

> [!IMPORTANT]
> **Status:** This project is currently **in development**. 

---

## 🌟 Core Functionalities

* **Isolated Workspaces**: Each organization registers for a dedicated environment to ensure its data remains private and secure.
* **Knowledge Base Ingestion**: Authorized users can upload PDF documents to build a custom library of company information.
* **Smart Assistant**: Employees can ask questions in natural language and receive answers based specifically on their company's documents.
* **Reliable Responses**: The system is designed to manage hallucinations by only answering from provided context and stating when information cannot be found.
* **Role-Based Access**: Multi-level permissions (Admin, Manager, Curator, and Employee) control who can manage users or upload documents.
* **Strict Data Isolation**: An infrastructure designed to ensure that one organization can never access the data of another.
